Creating a procastination app using django
steps
first we shall creat a folder in which our app will sit, using this code: mkdir 
then we shall creat an envirinment for our project using the venv code
after we shall activate the envirinment : venv\scripts\activate
then we shall install django in the envirinment using pip install django
after install django then we shall start a project using: django-admin startproject project name
then we shall start an app using python manage.py startapp appname
